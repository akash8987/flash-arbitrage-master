# Flash Arbitrage Master

This repository provides a professional-grade foundation for executing **Flash Loans** using the Aave V3 protocol to perform arbitrage across decentralized exchanges.

### Features
* **Aave V3 Integration:** Optimized for low-gas flash loan execution.
* **Uniswap V3 Swaps:** Pre-configured for multi-hop swaps.
* **Security:** Built-in ownership controls and emergency withdrawal functions.
* **Flat Structure:** Simplified for rapid deployment and auditing.

### Quick Start
1.  Configure your RPC URL in `.env`.
2.  Deploy `FlashArb.sol` to a supported EVM chain (Mainnet, Polygon, Arbitrum).
3.  Fund the contract with enough gas (ETH/MATIC) to cover transaction fees.
